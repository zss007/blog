vuex 之工具函数

2018-09-30
  • source

我们按照先易后难的原则，先把容易看懂的部分研究完，这节我们来看 ‘src/util.js’。

一、find

获取 list 执行 f 过滤后的第一个元素

二、deepCopy

递归调用深拷贝对象，cache 数组会缓存所有拷贝对象和嵌套对象，避免无限循环

三、forEachValue

遍历 obj，将键值对传入 fn 调用

四、isObject

判断是否为 Object

五、isPromise

判断是否为 isPromise

六、assert

断言，如果不满足 condition，则抛出 [vuex] ${msg} 错误
